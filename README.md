# CryptoSharp

## Overview

CryptoSharp is a C# collection of adapters for cryptocurrency exchanges.

## Features

### Swagger Definitions

Echange client are generated using [NSwag](https://github.com/RSuter/NSwag) based on definitions located in [doc/swaggers]()
